Archive pattern

This page simple serve as a demo, an example and a howto of repository extraction patterns

How it has been created

First you'll need to clean you working directory by storing all non commited changes (modified or untracked change) into a wip branch.

Then you can proceed with the following process 

# Creating an attic branch pointing at the root commit of the repository
git checkout --orphan attic
rm -rf *
git rm *
# create and edit index.html then pushing your content
git add index.html
git commit -m"Add the base page"
git push --set-upstream origin attic

# reference your newly created branch as a submodule
git checkout master
git submodule add -b attic htpps://github.com/tetedacier/tedacier.github.io attic
git add .
git commit -m"Reference attic branch in master working tree"
git push

This trigger a page build and the index is now accessible at `https://tetedacier.github.io/attic/`

If you need to update your `attic` folder content later, you'll need to do something like :

# checkout and cleanup attic directory
git checkout attic
rm -rf attic

# edit index.html and push its new content
git add index.html
git commit -m"Update the base page"
git push

# Update reference in master
git checkout master
git submodule update --init
cd attic
git pull
cd -
